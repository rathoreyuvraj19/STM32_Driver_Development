\doxysubsection{Detailed Description}
Priority encoding for STM32\+F4 NVIC (lower = higher priority). 

\begin{DoxyNote}{Note}
Cortex-\/\+M4 uses only upper 4 bits of priority byte. 

Priority 0 is highest urgency, 15 is lowest. 
\end{DoxyNote}
