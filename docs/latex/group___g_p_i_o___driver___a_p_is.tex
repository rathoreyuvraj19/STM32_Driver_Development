Configure a GPIO pin as an external interrupt (EXTI). 

This API configures the selected GPIO pin for external interrupt usage by\+:
\begin{DoxyItemize}
\item Mapping the GPIO port to the correct EXTI line using SYSCFG
\item Configuring the trigger type (rising/falling/both)
\item Enabling the interrupt line in EXTI
\item Setting the NVIC priority for the corresponding EXTI IRQ
\end{DoxyItemize}
